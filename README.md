# Finance Dashboard

A simple and responsive Finance Dashboard built using React.js.

## Features

* Dashboard with Balance, Income, Expense cards
* Line Chart for Spending Trend
* Pie Chart for Category Breakdown
* Transaction Table with details
* Role-based UI (Viewer / Admin)
* Insights section (Highest spending category)

## Tech Stack

* React.js (JavaScript)
* Bootstrap (Styling)
* Recharts (Data Visualization)

##  Project Structure

* Components: Sidebar, Dashboard, Transactions, Insights
* Data: Static transaction data

##  Run Locally

```bash
npm install
npm run dev
```
